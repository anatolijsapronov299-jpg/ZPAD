## Тема
C++ OpenCV.

## Опис роботи
У роботі створено модульну C++ програму з використанням OpenCV.

Програма:
- читає відео з камери;
- відображає відео у вікні;
- реагує на натискання клавіатури;
- реагує на дії миші;
- має декілька режимів обробки зображення;
- дозволяє змінювати параметр яскравості за допомогою слайдера;
- показує FPS;
- має scripts для встановлення, білду та запуску.

## Режими роботи

Клавіші:
- `0` — оригінальне зображення;
- `1` — інверсія кольорів;
- `2` — Gaussian blur;
- `3` — Canny filter;
- `4` — grayscale;
- `5` — sepia effect;
- `6` — glitch effect;
- `7` — cartoon effect;
- `8` — pixelation;
- `9` — picture in picture;
- `t` — показ або приховування текстової інформації;
- `q` або `Esc` — вихід з програми.

Миша:
- ЛКМ + перетягування — намалювати прямокутник;
- ПКМ — очистити намальовані прямокутники.

Слайдер:
- `Brightness` — змінює яскравість кадру.

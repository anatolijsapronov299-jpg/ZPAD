## Тема

Підготовка середовища, завантаження даних, очищення даних, формування вибірок та початковий аналіз даних у Python.

## Структура роботи

- `lab2_part1.ipynb` — робота з NOAA VHI/VCI/TCI даними по областях України.
- `lab2_part2.ipynb` — аналіз датасету Individual Household Electric Power Consumption.
- `requirements.txt` — список бібліотек для запуску ноутбуків.
- `README.md` — опис лабораторної роботи та інструкції запуску.

## Частина 1

У першій частині виконано роботу з VHI/VCI/TCI даними:

- створено структуру даних у форматі `pandas DataFrame`;
- виконано базове очищення даних;
- додано назви областей та індекси;
- реалізовано функції для вибірок:
  - ряд VHI для області за вказаний рік;
  - ряд VHI за діапазон років для вказаних областей;
  - пошук екстремумів, середнього та медіани;
  - додаткові вибірки на вимогу викладача;
- побудовано графік для порівняння значень VHI.

## Частина 2

У другій частині виконано аналіз датасету електроспоживання:

- завантажено та відкрито набір даних;
- виконано data cleaning;
- сформовано вибірки:
  - записи, де активна споживана потужність перевищує 5 кВт;
  - записи з силою струму в межах 19–20 А;
  - випадкова вибірка записів для обчислення середніх значень;
  - записи після 18:00 з високим рівнем споживання;
- виконано нормування та стандартизацію;
- розраховано коефіцієнти Пірсона та Спірмена;
- виконано One Hot Encoding категоріального атрибута;
- додано профілювання часу виконання процедур.

## Запуск

```bash
python -m venv .venv
```

Linux / macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

Встановлення залежностей:

```bash
pip install -r requirements.txt
```

Запуск Jupyter Notebook:

```bash
jupyter notebook
```

Після запуску потрібно відкрити `.ipynb` файли та виконати клітинки зверху вниз.

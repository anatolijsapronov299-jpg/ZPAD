## Опис роботи
Робота виконана на базі лабораторної роботи №6.  
До OpenCV-проєкту додано модуль `FaceDetector`, який виконує детекцію облич за допомогою OpenCV DNN та моделі ResNet SSD.

Програма:
- читає відео з камери;
- відображає кадри у вікні;
- має режими обробки зображення з лабораторної №6;
- реагує на клавіатуру та мишу;
- використовує OpenCV DNN для детекції облич;
- виконує детекцію облич у фоновому потоці;
- використовує `std::thread`, `std::mutex`, `std::condition_variable`, `std::atomic<bool>`;
- дозволяє вмикати / вимикати детекцію облич клавішею `F`;
- дозволяє вмикати / вимикати штучну затримку детектора клавішею `L`.

## Архітектура

- `CameraProvider` — робота з камерою через `cv::VideoCapture`.
- `FrameProcessor` — базова обробка кадрів та режими з лабораторної №6.

## Керування

Клавіші:

- `0` — оригінальне зображення;
- `1` — інверсія кольорів;
- `2` — Gaussian blur;
- `3` — Canny filter;
- `4` — grayscale;
- `5` — sepia;
- `6` — glitch effect;
- `7` — cartoon effect;
- `8` — pixelation;
- `9` — picture in picture;
- `F` — увімкнути / вимкнути face detection;
- `L` — увімкнути / вимкнути штучну затримку детектора;
- `T` — показати / приховати текст;
- `Q` або `Esc` — вихід.

Миша:

- ЛКМ + перетягування — намалювати прямокутник;
- ПКМ — очистити прямокутники.

Слайдер:

- `Brightness` — змінює яскравість кадру.

## Логіка потоків

### Main Thread
1. Читає кадр з камери.
2. Передає копію кадру в `FaceDetector`.
3. Забирає останні відомі координати облич.
4. Малює ці координати поверх кадру.
5. Показує результат через `cv::imshow`.

### Worker Thread
1. Чекає на новий кадр через `std::condition_variable`.
2. Коли кадр отримано, запускає `net.forward()`.
3. Фільтрує результати за `confidence > 0.5`.
4. Зберігає координати облич у спільну змінну під `std::mutex`.

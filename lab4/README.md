## Тема
Візуалізація даних 2.

## Опис роботи
У роботі створено інтерактивну програму для візуалізації гармоніки виду:

```text
y(t) = A * sin(ωt + φ)
```

Програма дозволяє:
- змінювати амплітуду;
- змінювати частоту;
- змінювати фазовий зсув;
- додавати шум;
- змінювати середнє значення шуму;
- змінювати дисперсію шуму;
- показувати або приховувати шум;
- фільтрувати зашумлену гармоніку;
- скидати параметри до початкових значень.

## Структура
- `lab4.ipynb` — основний Jupyter Notebook з кодом лабораторної роботи.
- `requirements.txt` — потрібні бібліотеки.

## Встановлення
```bash
python -m venv .venv
source .venv/bin/activate      # Linux / macOS
# або
.venv\Scripts\activate       # Windows

pip install -r requirements.txt
```

## Запуск
```bash
jupyter notebook
```

Після запуску відкрийте `lab4.ipynb` та виконуйте клітинки зверху вниз.

## Використання
1. Запустіть усі клітинки ноутбука.
2. Змінюйте параметри гармоніки за допомогою слайдерів.
3. Увімкніть або вимкніть показ шуму за допомогою чекбокса `Show Noise`.
4. Змініть частоту зрізу фільтра слайдером `Cutoff Frequency`.
5. Натисніть `Reset`, щоб повернути початкові параметри.

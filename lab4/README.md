## Тема

Інтерактивна візуалізація гармонічної функції з накладеним шумом та фільтрацією.

## Мета роботи

Метою лабораторної роботи є створення програми, яка дозволяє користувачу будувати графік гармоніки виду:

```text
y(t) = A * sin(ωt + φ)
```

з накладеним шумом, змінювати параметри гармоніки та шуму за допомогою інтерактивних елементів, фільтрувати шум і порівнювати результати.

## Виконані завдання

У роботі реалізовано:

1. Функцію гармоніки, яка приймає параметри:
   - `amplitude` — амплітуда;
   - `frequency` — частота;
   - `phase` — фазовий зсув;
   - `noise_mean` — середнє значення шуму;
   - `noise_covariance` — дисперсія шуму;
   - `show_noise` — прапорець для показу шуму.

2. Інтерактивний інтерфейс з елементами:
   - графік функції;
   - sliders для зміни амплітуди, частоти, фази, середнього шуму, дисперсії шуму та параметра фільтра;
   - checkbox для показу або приховування шуму;
   - кнопка `Reset` для повернення початкових параметрів.

3. Логіку оновлення графіка:
   - при зміні параметрів гармоніки оновлюється гармоніка;
   - при зміні параметрів шуму оновлюється шум;
   - якщо змінюються тільки параметри гармоніки, шум залишається таким самим;
   - якщо змінюються параметри шуму, шум генерується заново.

4. Фільтрацію зашумленої гармоніки:
   - використано ковзне середнє як простий фільтр;
   - побудовано початкову гармоніку, зашумлену гармоніку та відфільтровану гармоніку.

5. Інструкцію користувача.

## Інструкція користувача

- Змініть значення `Amplitude`, щоб збільшити або зменшити висоту гармоніки.
- Змініть `Frequency`, щоб змінити кількість коливань.
- Змініть `Phase`, щоб зсунути графік по осі часу.
- Змініть `Noise Mean` або `Noise Covariance`, щоб змінити шум.
- Використайте `Show Noise`, щоб показати або приховати зашумлений сигнал.
- Змініть `Filter Window`, щоб змінити ступінь згладжування.
- Натисніть `Reset`, щоб повернути початкові параметри.
